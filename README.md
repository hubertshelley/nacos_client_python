# nacos_client_python

#### 介绍
服务于**nacos**的自动化服务注册插件

#### 软件架构
推荐使用py3.7或以上环境使用  
基本上是模块化的Nacos open-api,使用参数与open-api一致。 
并提供了nacos实例的自动心跳包功能


#### 安装教程
源码安装
```
git clone https://gitee.com/hubert22/nacos_client_python.git
cd nacos_client_python
python setup.py install
```

#### 使用说明

请参照client_test运行
